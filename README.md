# 🍁 Kids Canada Trivia

A single-player Canada trivia quiz game for Grade 5 students. Runs entirely in the browser — no server or installation needed.

## Play

Open `index.html` in any web browser.

## Features

- 50 questions covering Geography, History, Culture, Science & Nature
- Choose 10, 20, or all 50 questions
- 20-second timer per question
- Correct/wrong feedback after each answer
- Score summary with question breakdown at the end

## Host on GitHub Pages

1. Go to **Settings → Pages** in this repo
2. Set Source to your branch, folder `/` (root)
3. Save — your game will be live at `https://<username>.github.io/kids-canada-trivia/`
